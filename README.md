## pm2

```cmd
pm2 start filename
```

파일 실행

```cmd
pm2 start main.js --watch --ignore-watch="data/* sessions/*"  --no-daemon
```

pm2를 실행하면서 로그가 출력되도록 합니다. (--no-daemon) 또 특정 디랙토리에 대한 watch를 하지 않도록 하는 방법
